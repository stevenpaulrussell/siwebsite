Instructions

Welcome to Carepost. Use this page to set up a 'viewer', or to get guidance on use.

If you have already made your first postcard, go here: set up a viewer.

To make your firsr postcard, read further:

  • Start by texting a photo to one of the three current Carepost numbers: +1 415 779 7677, +1 832 662 6430, or +1 830 521 5373.
  • Instructions sent back should lead you through. There will be a delay while the server spins up. (I'm keeping costs down).
  • On your phone, please be sure to make a unique contact name for that number so that it is convenient to access!
  • Having made your first postcard, text the command 'connector' to your new contact. You will get a 4-character code.
  • That 4 character code is used in the link above to make a viewer for acual use including just seeing how this works.
  • Or, you use that code to send to an existing viewer set up by someone else, see below.

    • If you want to enable another person to send postcards to the viewer you set up, here is how:

  • First, have them get their own 4-character code by texting the command 'connector' to their own Carepost contact, as noted.
  • You will enter that code and their own phone number -- the phone number that you can call them on -- in a command texted to Carepost.
  • The command is:
  • connect connector <4-character code>
  • That's it!. Their profile image will appear on the viewer you set up.
